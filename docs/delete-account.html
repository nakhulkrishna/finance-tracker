Delete your Finance Tracker account.

This page can be used as your App Store account deletion URL. It explains how a user can request deletion and what to expect when their Finance Tracker account is removed.

Request account deletion Open support

What gets removed

  • Profile details linked to your account
  • Home operations and balance records
  • Wallet and investment entries
  • Premium billing mirror data stored for app state
How it works

Keep the deletion path simple and transparent.

1. Send a deletion request

Contact your official support email and use the subject line Delete My Finance Tracker Account.

2. Include account details

Ask the user to include the email address used in Finance Tracker so the right account can be found.

3. Confirm identity if needed

If support needs confirmation, ask the user to reply from the same email linked to their Finance Tracker account.

4. Remove cloud-backed app data

Delete the user’s account data stored for profile, finance records, wallet activity, investments, and app-linked preferences.
